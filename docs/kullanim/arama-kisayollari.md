# Arama Kısayolları (Adres Çubuğu)

## Nedir?
JahBrowser adres çubuğuna kısa bir anahtar kelime yazıp arama yaptığında seni doğrudan ilgili sitenin arama sonuçlarına götürür. Ayrı ayrı sitelere gidip aramana gerek kalmaz. Bu kısayollar **hazır gelir**, kurulum gerektirmez.

## Hazır kısayollar

| Yaz | Sonuç |
|---|---|
| `kick <sorgu>` | Kick'te arama (kanal/yayın) |
| `yt <sorgu>` | YouTube'da arama |
| `tw <sorgu>` | Twitch'te arama |
| `gh <sorgu>` | GitHub'da arama |
| `wiki <sorgu>` | Türkçe Vikipedi'de arama |

## Nasıl kullanılır?
1. Adres çubuğuna kısayolu yaz: örneğin `yt`.
2. **Boşluk** veya **Tab** tuşuna bas — adres çubuğu o aramaya geçer ("YouTube Arama" görünür).
3. Aramak istediğin şeyi yaz: `yt` + Tab + `jahrein klip`.
4. **Enter**'a bas — doğrudan YouTube arama sonuçları açılır.

**Örnekler**
- `kick` + boşluk + `jahrein` → Kick'te "jahrein" araması.
- `yt` + boşluk + `en iyi klipler` → YouTube arama sonuçları.
- `wiki` + boşluk + `İstanbul` → Türkçe Vikipedi.

## İpuçları
- Kick birincil olduğu için `kick` kısayolu ilk sıradadır.
- Kısayolları görmek/düzenlemek için: **Ayarlar → Arama motoru → Arama motorlarını yönet.**
- İstersen kendi kısayollarını da ekleyebilirsin (aynı ayar sayfasından).
